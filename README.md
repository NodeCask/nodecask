# NodeCask

---

## 📖 Introduction

NodeCask 是一个简约的社区程序！

## ✨ Features

- 后端由 Rust 编写
- 数据存储使用 SQLite3 

## 🚀 Getting Started

运行下面命令，然后浏览器打开 `http://127.0.0.1:3000` 安装提示设定管理员信息即可。

```bash
docker run -d --name nodecask -p 127.0.0.1:3000:3000 nodecask/nodecask:latest
```
